# Etude des régresseurs de calendriers

Le but de ce projet est d'étudier les possibles divergences entre les regresseurs de calendriers générés par les programmes SAS (écrit par Dominique LADIRAY) et par les nouveaux packages du rjduniverse (*rjd3modelling*).

Cette étude cherche aussi dans une plus importante mesure à essayer de comprendre comment sont créés les régresseurs de calendriers.

## Résultat et production

Après l'étude nous avons rédigé 2 types de documents :
- des programmes R qui reproduisent la méthode de création des regresseurs SAS et par RJD
- une petite documentation des programmes et de la théorie des régresseurs de calendrier
